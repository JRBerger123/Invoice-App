## Features

- Calculate total cost based on item description, quantity, and price.
- Apply discounts based on subtotal.
- Display a detailed receipt with itemized costs and total savings.

## Installation

To install and run this project locally, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/JRBerger123/Invoice-App.git
    ```
2. Navigate to the project directory:
    ```sh
    cd Invoice-App
    ```
3. Compile the Java source files:
    ```sh
    javac src/InvoiceApp.java -d bin
    ```
4. Run the application:
    ```sh
    java -cp bin InvoiceApp
    ```

## Usage

1. Run the application.
2. Enter the item description, quantity, and price when prompted.
3. Continue adding items until you are done.
4. View the detailed receipt with total cost and savings.
